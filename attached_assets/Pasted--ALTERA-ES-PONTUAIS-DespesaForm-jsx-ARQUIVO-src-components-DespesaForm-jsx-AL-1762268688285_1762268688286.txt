### **ALTERAÇÃO 3: Adicionar após linha 340 (Nova Função)**

**✅ ADICIONAR ESTA FUNÇÃO:**

```javascript
  // ✅ NOVA FUNÇÃO: Determinar se despesa deve ser marcada como EXECUTADA
  const determinarStatusExecucao = useCallback((dados) => {
    // Se tem dados de empenho/liquidação/pagamento → EXECUTADA
    if (
      dados.numeroEmpenho || 
      dados.dataEmpenho || 
      dados.dataLiquidacao ||
      dados.dataPagamento ||
      dados.numeroNota
    ) {
      return "EXECUTADA";
    }
    
    // Caso contrário → PLANEJADA
    return "PLANEJADA";
  }, []);
```

## ⚠️ IMPORTANTE

A lógica é:
1. **Criar despesa** → `status: "PLANEJADA"`, `statusPagamento: "pendente"`
2. **Adicionar dados de empenho** → `status` automaticamente vira `"EXECUTADA"`
3. **Status de pagamento** → Controlado separadamente pelo campo `statusPagamento`
