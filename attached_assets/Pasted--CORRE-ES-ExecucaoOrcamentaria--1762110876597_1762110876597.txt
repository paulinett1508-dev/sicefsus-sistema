❌ PROBLEMA: Modal não abre ao clicar em "▶️ Executar"

SOLUÇÃO:
1. Verifique o console:
   - Se aparecer o log "▶️ ExecucaoOrcamentaria: Executando..."
     → Handler está funcionando, problema é no render do modal
   - Se NÃO aparecer o log
     → Problema no botão, verifique linha ~690

2. Se handler funciona mas modal não abre:
   - Verifique se ExecutarDespesaModal está importado (linha ~33)
   - Verifique se modal está sendo renderizado (CORREÇÃO #3)
   - Verifique state `executandoDespesa` no console
