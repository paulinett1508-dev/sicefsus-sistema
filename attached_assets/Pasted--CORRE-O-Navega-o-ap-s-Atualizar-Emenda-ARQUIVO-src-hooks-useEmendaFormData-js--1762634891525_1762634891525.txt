# 🔧 CORREÇÃO - Navegação após "Atualizar Emenda"

## 📍 ARQUIVO: `src/hooks/useEmendaFormData.js`

## 🎯 PROBLEMA
Ao clicar em "Atualizar Emenda", o sistema mostra toast de sucesso mas **não volta para a listagem**.

## ✅ SOLUÇÃO

### **Localizar esta seção (linha ~565):**

```javascript
if (isEdicao) {
  await updateDoc(doc(db, "emendas", id), dadosParaSalvar);
  setToast({
    show: true,
    message: "✅ Emenda atualizada com sucesso!",
    type: "success",
  });
  // 🔄 EDIÇÃO: NÃO navegar, deixar usuário na emenda
  // Apenas mostra toast de sucesso
}
```

### **SUBSTITUIR POR:**

```javascript
if (isEdicao) {
  await updateDoc(doc(db, "emendas", id), dadosParaSalvar);
  setToast({
    show: true,
    message: "✅ Emenda atualizada com sucesso!",
    type: "success",
  });
  
  // ✅ NAVEGAÇÃO: Voltar para listagem após atualizar
  setTimeout(() => {
    navigate("/emendas");
  }, 1500); // Aguarda 1.5s para usuário ver o toast
}
```

---

## 📝 EXPLICAÇÃO

### **ANTES:**
```
1. Usuário edita emenda
2. Clica "Atualizar Emenda"
3. Toast de sucesso aparece
4. ❌ Fica na mesma tela (não navega)

### **DEPOIS:**
```
1. Usuário edita emenda

3. Toast de sucesso aparece
4. ✅ Aguarda 1.5 segundo

if (isEdicao) {

## ⚙️ OPÇÃO AVANÇADA: Perguntar ao Usuário

Se quiser dar opção ao usuário (ficar ou voltar):

```javascript
if (isEdicao) {
  await updateDoc(doc(db, "emendas", id), dadosParaSalvar);
  
  // Modal personalizado
  const resposta = window.confirm(
    "✅ Emenda atualizada com sucesso!\n\n" +
    "Deseja voltar para a listagem de emendas?"
  );
  
  if (resposta) {
    navigate("/emendas");
  } else {
    setToast({
      show: true,
      message: "✅ Emenda atualizada! Você pode continuar editando.",
      type: "success",
    });
  }
}
```

## 🚀 IMPLEMENTAÇÃO RECOMENDADA

**Use a primeira opção (com setTimeout)** porque:
- ✅ Usuário vê o toast de sucesso
- ✅ Tem tempo de perceber que salvou
- ✅ Navegação suave
- ✅ Experiência consistente com criação de emenda
