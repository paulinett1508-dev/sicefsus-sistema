# Correção de Problema

Para o problema descrito:

## 1. Diagnóstico
- Identifique a causa raiz
- Liste arquivos envolvidos
- Trace o fluxo de dados

## 2. Solução
- Descreva a correção necessária
- Mostre ANTES/DEPOIS do código
- Liste efeitos colaterais possíveis

## 3. Implementação
- Forneça o código corrigido COMPLETO
- Nunca fragmente a solução
- Inclua validações necessárias

## 4. Teste
- Descreva como verificar se funcionou
- Liste casos de borda a testar
